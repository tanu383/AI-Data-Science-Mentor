# 🤖 AI Data Science Mentor

An AI-powered chatbot built using **Google Gemini API**, **Python**, and **Streamlit** to help students learn Data Science, Machine Learning, Deep Learning, Python, DSA, and AI concepts in an interactive way.

---

## 🚀 Features

- 💬 Interactive AI Chatbot
- 🤖 Powered by Google Gemini AI
- 📚 Answers Data Science and AI-related questions
- 🐍 Python Programming Guidance
- 🧠 Machine Learning & Deep Learning Support
- 📊 Data Analytics Concepts
- 💼 Career Guidance for AI/ML Aspirants
- 🌙 Modern Dark-Themed UI
- 🔒 Secure API Key Management using `.env`

---

## 🛠️ Tech Stack

- Python
- Streamlit
- Google Gemini API
- python-dotenv

---



---

## 🎯 Use Cases

- Learning Python
- Understanding Machine Learning concepts
- Deep Learning guidance
- Data Science interview preparation
- DSA-related queries
- Career roadmap disc

---

## 🔒 Security

API keys are stored in a `.env` file and are not included in the public repository.

---

## ⚠️ Disclaimer

This project uses Google's Gemini AI model. Responses generated by the AI may occasionally contain inaccuracies. Users should verify important information independently.
